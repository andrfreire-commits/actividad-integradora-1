# Proyecto Flutter Básico

Aplicación Flutter básica desarrollada como práctica académica.

## Funcionalidades
- Pantalla principal con AppBar personalizado
- Botón funcional con SnackBar
- Navegación a segunda pantalla
- Botón para regresar a la pantalla principal
- Uso del widget SwitchListTile investigado de forma autónoma

## Nota
Durante el desarrollo se presentaron problemas técnicos de red que impidieron la descarga del Flutter SDK y la ejecución local del proyecto.  
No obstante, el código fuente cumple con todos los requerimientos funcionales solicitados y se encuentra estructurado.
